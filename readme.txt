== Description ==

This plugin lets you embed Github's Gists in your posts or pages, but the main deference then all others is that this plugin also enables you to create Gists without leaving your WordPress.
that's right you can create gists from within your WordPress admin.
[youtube http://www.youtube.com/watch?v=Q_dBkGQSU7g]

**Features**

*	Updated to the latest GitHub API V3
*	Create Gists from your WordPress Admin panel
*	Internal cache built-in
*	Built in Tinymce (editor) button.
*	Works with all posts types (post,page,custom).


any feedback or suggestions are welcome.

check out my [other plugins][1]


[1]: http://en.bainternet.info/category/plugins


== Installation ==
Simple steps:  

1.  Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation.
1.  Then activate the Plugin from Plugins page.
1.  Done!
